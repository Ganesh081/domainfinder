# Domain Name Generator

A premium, interactive domain name generator tool that helps you find brandable domain names across multiple extensions.

## Features

- 🎯 **Smart Name Generation** - Creates brandable domain suggestions based on your niche
- 🌐 **Multiple Extensions** - Searches across .com, .in, .xyz, .ai, and .site domains
- ⭐ **Organized Results** - Results grouped by extension with conversion-optimized labels
- ⚡ **Instant Results** - Fast domain generation with smooth animations
- 📱 **Responsive Design** - Works perfectly on desktop and mobile devices
- 🎨 **Premium UI** - Modern SaaS-style interface with excellent UX
- 💸 **Conversion Optimized** - Clear pricing indicators and prominent call-to-action buttons

## How to Use

1. Open `index.html` in your web browser
2. Enter your niche or business idea (e.g., "AI Tools", "Social Media")
3. Click "Generate" or press Enter
4. Browse results organized by domain extension
5. Click the "Buy" button to register a domain on Namecheap

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Grid, animations
- **Vanilla JavaScript** - No frameworks or dependencies

## Domain Extensions

- **⭐ .com** - Recommended (most popular)
- **🇮🇳 .in** - Indian local domains
- **💸 .xyz** - Cheapest option
- **🤖 .ai** - AI-focused brands
- **🌐 .site** - Broader category
